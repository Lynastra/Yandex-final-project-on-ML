# Детекция дипфейков (Deepfake Detection)

Финальный проект курса по машинному обучению. Задача — бинарная классификация изображений: определить, является ли лицо на фото реальным или сгенерированным (дипфейком).

## Структура репозитория

| Файл | Описание |
|------|----------|
| `ml_проект_дипфейки_TTA-2.ipynb` | Основной ноутбук с загрузкой данных, обучением модели и инференсом |
| `best_model.pth` | Веса лучшей обученной модели (PyTorch) |
| `отчёт_по_финальному_проекту.pdf` | Итоговый отчёт с описанием методологии и результатов |
| `.gitattributes` | Конфигурация Git LFS для хранения больших файлов |

## Использование

### Клонирование репозитория
Так как модель хранится через Git LFS, клонировать нужно с подгрузкой больших файлов:
```bash
git lfs clone https://github.com/Lynastra/Yandex-final-project-on-ML.git
```

## Технологии

- Python 3.x
- PyTorch
- Torchvision
- NumPy, Pandas
- Matplotlib, Seaborn
- Git LFS
